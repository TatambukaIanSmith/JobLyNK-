🔧 How It Works

1. Navigation: Click "Applications" in the employer dashboard sidebar

2. Statistics: See overview cards showing application counts by status

3. Search & Filter: Use the search box and dropdown filters to find specific applications

4. Application Management: Use action buttons to:

  • • View Details: See full application information
  • • Approve: Accept the application (sends notification to worker)
  • • Reject: Decline the application (sends notification to worker)
  • • Contact: Send message to the applicant

5. Real-time Updates: Actions update the database and refresh the view
